# Vinyl Tracker — Product Roadmap

## MVP (complete)
- [x] Add records to collection (artist, album, year, genre, artwork URL, star rating, favourite track, cost, notes)
- [x] Wishlist — records you want to buy
- [x] Spend tracking — total cost across collection
- [x] New releases page via API
- [x] Favourite upcoming releases

---

## In progress
- [x] Visual redesign
  - Dark mode first design (bg #0a0a0a, surface #111111)
  - Geist font
  - Sidebar navigation
  - Grid and list view toggle
  - Stat pills (total records, total spent, average rating)
  - Album artwork as hero in cards
  - Skeleton loaders
  - Genre pills, empty states, hover transitions

---

## Sprint 2 — Engagement
- [ ] Discogs autofill — search album/artist to auto-populate all fields including artwork
- [ ] Stats dashboard — spend over time, records by genre, top artists, average rating
- [ ] Search and filter — filter collection by genre, artist, year, rating

---

## Sprint 3 — Depth
- [ ] Listening log — log when you played a record with date and notes
- [ ] Custom shelves — organise records into user-defined groups (moods, genres, playlists)
- [ ] Condition grading — Mint, VG+, VG, Good per record

---

## Sprint 4 — Value
- [ ] Collection value tracker — estimated market value via Discogs pricing data
- [ ] Import / export — CSV backup and restore
- [ ] Shareable profile — public link so others can browse your collection

---

## Future ideas (backlog)
- Mobile app (React Native)
- Barcode scanner for adding records
- Last.fm integration for listening stats
- Recommendations based on your collection
- Friends / social features
